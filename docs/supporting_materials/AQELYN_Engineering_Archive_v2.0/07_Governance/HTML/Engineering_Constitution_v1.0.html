# AQELYN Engineering Constitution v1.0

Mission: Build the world's most understandable, evidence-driven, extensible and trusted Cyber Security Operating Environment.

Core articles:
1. Simplicity before complexity.
2. Evidence before opinion.
3. Security must be explainable.
4. The user is always in control.
5. Accessibility is a requirement.
6. Architecture before code.
7. Modular forever.
8. Documentation is part of the product.
9. Testing is mandatory.
10. Python first.
11. Long-term thinking.
12. Everything is traceable.
13. Build trust.
14. The Calm Principle: reduce fear and increase clarity.
