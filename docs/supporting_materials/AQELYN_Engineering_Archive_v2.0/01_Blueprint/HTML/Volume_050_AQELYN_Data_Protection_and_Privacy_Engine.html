Volume 50
AQELYN Data Protection & Privacy Engine (SDPPE)
Status: Core Security Domain
Vision
AQELYN must protect not only systems, but also the data inside and around
those systems.
The platform should answer:
•  What sensitive data exists?
•  Where is it stored?
•  Who can access it?
•  Is it encrypted?
•  Is it backed up?
•  Is it exposed?
•  Is it governed by policy?
•  Is privacy respected?
Data Philosophy
Data is not just files.
Data includes:
Documents
Every data source becomes part of the Digital Twin where appropriate.
Data Object
Every important data collection can become a AQELYN Object.
Data Object
Data Classification
Initial classifications:
Public
Organizations can define custom classifications.
Privacy by Design
AQELYN must minimize unnecessary collection.
The platform should collect:
•  only what is needed,

Sensitive Data Awareness
The engine can help identify categories such as:
•  Personal information
•  Credentials
•  API keys
•  Financial information
•  Health information
•  Legal documents
•  Customer data
•  Security evidence
•  Internal reports
Detection must be careful, configurable, and evidence-based.
Encryption Verification
The engine checks:
•  Disk encryption
•  Database encryption
•  Backup encryption
•  Cloud storage encryption
•  Transport encryption
•  Key management status
Access Intelligence
The platform answers:
Who can access this data?
Data Retention
Organizations define retention policies.
Examples:
•  Evidence retained for 12 months.
•  Incident data retained for 7 years.
•  Personal data minimized.
•  Reports archived after review.
•  Logs rotated according to policy.

Data Protection Is a First-Class Security Domain
Status: Accepted
Decision
Project  AQELYN  shall  treat  data,  privacy,  retention,  encryption,  backup,  access,  and
exposure as first-class parts of the Cyber Security Operating Environment.
Rationale
Security  cannot  be  measured  only  by  device  or  network  posture.  The  true  objective  is
protecting people, organizations, operations, and data.
Strategic Principle
AQELYN protects systems so that people and data remain protected.
