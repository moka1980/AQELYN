# Project Charter Addendum v2.1 - Endpoint and Exposure Expansion

This addendum confirms that AQELYN includes authorized private endpoint security assessment, enterprise endpoint telemetry, web intelligence, attack surface discovery, vulnerability intelligence and canonical asset inventory.

Mandatory controls:
- Authorization and scope are required before assessment.
- Private endpoint assessment must be consent-based and privacy-preserving.
- Web and attack surface discovery must default to passive-first collection and use active probing only when configured.
- All findings must be evidence-backed and traceable.
- The fixed repository structure remains immutable.
