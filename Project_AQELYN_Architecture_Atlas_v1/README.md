# Project AQELYN Architecture Atlas v1.0

This package contains a readable vector architecture atlas for Project AQELYN.

## Contents

- `pdf/Project_AQELYN_Architecture_Atlas_v1.pdf` - PDF atlas with 25 diagrams plus cover.
- `html/index.html` - browsable HTML atlas.
- `diagrams/*.svg` - individual vector diagrams.
- `poster/Project_AQELYN_Color_Overview_Poster.svg` - colorful one-page overview poster.
- `manifest.json` - file inventory with SHA-256 hashes.

## Purpose

The atlas explains how the whole Project AQELYN program should be built: engineering archives, dependencies, coding order, runtime architecture, endpoint/web/security engines, deployment, CI/CD, and operations.

The diagrams are SVG/vector based so text remains crisp when zoomed, printed, or embedded in PDF/HTML documentation.
